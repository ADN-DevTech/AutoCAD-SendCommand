There are three commands in this demo:

1. TestSimple
   Run it directly, it will send command to draw a line with complete set of command tokens

2. TestCallback
  Run it directly, it will send command to draw a line with incomplete set of command tokens, by traditional way: callback function

3.TestAwait
  Run it directly, it will send command to draw lines with incomplete set of command tokens, by .NET 4.5 feature: await and async
