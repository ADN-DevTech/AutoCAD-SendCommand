There are three commands in this demo, testing in AutoCAD 2014

1. TestGetUserInput: 
  1) open two drawings.
  2) in one drawing, run the command, it asks you to input a point.
  3) do not input the point, while switch to the other drawing 
  4) switch back to the previous drawing, you can find your command is still running

2. TestSendCommand1
  Run it directly, it will send command to draw a line with complete set of command tokens

3.TestSendCommand2
  Run it directly, it will send command to draw lines with incomplete set of command tokens.
